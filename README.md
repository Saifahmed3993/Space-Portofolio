## 🚀 Getting Started

```bash
# Clone the repo
git clone https://github.com/Saifahmed3993/Space-Portofolio.git
cd Space-Portofolio

# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
