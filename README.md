## ⚡ Performance Optimisations

- 🎯 **`next/dynamic`** lazy-loading for Three.js, About, Skills & Projects — instant first paint
- 🖼️ **AVIF + WebP** auto-conversion via `next/image` — up to 60% smaller images
- 🎬 **`preload="none"`** on all background `.webm` videos — no blocking downloads
- ⭐ **Three.js star count** reduced 5 000 → 2 000 + `dpr` capped at 1× — smoother GPU
- 🖱️ **`pointer-events: none`** on the canvas — all links and buttons fully clickable
- 🗜️ **Brotli compression** + 60-day image cache TTL

## 🚀 Getting Started

```bash
# Clone the repo
git clone https://github.com/Saifahmed3993/Space-Portofolio.git
cd Space-Portofolio

# Install dependencies
npm install

# Run the dev server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
